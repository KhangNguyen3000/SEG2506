Handle Arcs

HISim implements two extra type of arcs besides the regular -resource- arcs:

  • Test arcs: the basic functioning is as follows; if a test arc connects a certain place with a discrete transition, the place's marking will be compared with the arc weight for the place's enabling as if it was a regular arc, but when the transition fires, no tokens -or continuous amout- will be removed from the input place. On the other hand, if a test arc is connected to a continuous transition, the input place's marking can be used as a variable in the transition's firing function, but the arc will have no effect otherwise. Test arcs can't create conflicts.
  • Inhibitory arcs: the functioning is similar to a test arc in that they are used for enabling purposes but not for token or continuous amount flow. If the marking of the corresponding input place is greater or equal than the arc wegight, then the transition won't be enabled.

To add an arc you can click on this  icon in the toolbar. An arc can be directed from a place to a transition and visa-versa by left clicking the corresponding place and transition. A arc may have angles by clicking at any intermediate point between the place and the transition. If the shift key is hold at the same time the arc will be appear in a shape of a curve instead in a shape of an angle.

 

To change the type of an arc or other properties, you can right click on it and then select an option in the pop-up menu.
