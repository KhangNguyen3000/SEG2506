Handle Continuous Places

A continuous place's marking is a real number, positive, negative or 0, and they don't have a Capacity property.

To add a continuous place you can click on this   icon in the toolbar, or select the Continuous Place option in the Draw menu. Then left clicking at any position in the drawing area a new continuous place will be created there.

 

To change the properties of a continuous place, you can right click on it and then select an option in the pop-up menu.
