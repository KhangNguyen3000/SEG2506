Handle Discrete Places

A discrete place's marking is a positive integer or 0. If the Capacity property of a discrete place is set to a certain number (see how to set properties below), the marking will then be limited to integers from 0 to that number. A capacity of 0 represents no limitation, or an infinite capacity.

To add a discrete place you can click on this   icon in the toolbar, or select the Discrete Place option in the Draw menu. Then left clicking at any position in the drawing area a new discrete place will be created there.

To add a token, i.e. a unit to the marking of a discrete place you can click on this   icon in the toolbar, then left clicking on a discrete place will add a token to it. To remove a token you will do a similar process but clicking on this   icon instead.

 

To change this and other properties of a discrete place, you can right click on it and then select an option in the pop-up menu.
