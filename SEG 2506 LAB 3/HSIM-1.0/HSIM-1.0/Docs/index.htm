HISim help

HISim allows you to create and simulate Hybrid Petri nets. You can use HISim in Draw mode or in Simulation mode, but not both simultaneously. To switch between the two modes you can click on this  icon in the toolbar, or select the Simulation mode in the Simulate menu.

HISim implementation of Hybrid Petri nets follows -with a few additional features- Demongondin and Koussoulas' definition of Differential Petri nets. See "Demongodin, I. and Koussoulas, N.T. 1998. Differential Petri Nets: Representing Continuous Systems in a Discrete-Event World. IEEE Transactions on Automatic Control, Vol. 43, No. 4, 573-579".
